What is Palladium?

Palladium is a Bitcoin fork that uses the same cryptographic algorithms for transactions, ensuring proven security and reliability. With an average block time of just 2 minutes, transaction validation is significantly faster than Bitcoin's 10 minutes.

Maximum supply of 21,000,000 PLM with a halving every 210,000 blocks. Network security is maintained through the SHA-256 proof-of-work algorithm — the same one securing Bitcoin.
